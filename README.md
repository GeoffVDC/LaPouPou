# La Pou Pou
Discord utility bot

## Media Commands
- `!join` - join your channel
- `!play` - play a song
- `!pause` - pause the current song
- `!stop` - stop playing audio
- `!disconnect` - leave the discord server
- `!skip` - skip current song
- `!clear` - clear the queue
- `!ping` - check latency
- `!hoya` - ask for a hoya
